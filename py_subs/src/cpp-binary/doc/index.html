Binary star software home hage

The binary star software provides a small set of generally useful functions for binary stars (circular orbits only I'm afraid). Thus it has functions for computing orbital periods and separations, the angular momentum loss rate due to gravitational waves and a few associated routines to do with stellar radii. There are no programs, only callable functions. The software is written in C++. See the API docs for detailed information upon the available functions.

Installation

To install, download the gzipped tar file ,gunzip and tar xvf it. A new directory will be created. Change directory into it and read the README file. Get in touch with me if you have problems.
